1. Cara Download dan Load File .zip ke Qt Creator
   - Klik nama file .zip nya di Github
   - pergi ke kanan dan tekan tombol download (download raw file)
   - Unzip file .zip nya
   - Buka Qt Creator nya
   - Open Project dan pergi ke folder yang udah diunzip
   - Klik file .txt nya
   - Projek berhasil diload di Qt Creator
